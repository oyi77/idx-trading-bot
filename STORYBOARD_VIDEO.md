## 📹 Visual Style Guide

### Color Palette
- **Primary**: Emerald Green (#10b981) — bullish, action
- **Secondary**: Indigo (#6366f1) — tech, AI
- **Accent**: Amber (#f59e0b) — premium tier
- **Background**: Black (#000000) with subtle mesh gradient
- **Text**: White (#f1f5f9), muted gray for secondary

### Animations
- **Entrance**: Fade + scale (0.3s ease-out)
- **Transitions**: Smooth 0.5s fade
- **Data reveals**: Slide-in from left with slight bounce
- **Emphasis**: Subtle glow/pulse on key metrics
- **Text**: Typewriter effect for commands (optional, 0.1s per char)

### Font Usage
- **Headlines**: Inter Bold (700)
- **Body**: Inter Regular (400)
- **Numbers/Tickers/Commands**: JetBrains Mono (400)
- **Sizes**: H1=48px, H2=32px, Body=16px, Mono=14px

### Icon Source
- Lucide React icons (minimalist, clean)
- Brain, TrendingUp, Zap, Bell, BarChart3, Rocket
